# generator-heirloom

[Yeoman](http://yeoman.io/) generator for [heirloom-seed](https://github.com/xuyuanxiang/heirloom-seed) (the template project).

## Installation
```bash
npm install yeoman generator-heirloom -g
```

## Usage

create your project:

```bash
yo heirloom your-project-name
```
